HANDLEIDING / MANUAL (NL/EN)

1. Songs & Database

NL: Beheer hier je liedjes. Klik op + voor toevoegen aan set.
EN: Manage your songs. Click + to add to setlist.

2. Bulk Toevoegen & Tagging

NL: Klik op SELECT in het SONG menu. Vink meerdere liedjes aan. De knop onderin verandert naar "ZET X IN SET". Klik hierop om de hele selectie in één keer naar je setlist te verplaatsen. Tik op een tag in de cloud om deze aan de selectie toe te voegen.
EN: Click SELECT in the SONG menu. Check multiple songs. Use the bottom button "ZET X IN SET" to move all to setlist at once. Tap a tag in the cloud to apply to selection.

3. WhatsApp Import/Export

NL: Ga naar SET > SET BEHEER. Gebruik WA-EXPORT om de lijst naar klembord te kopiëren. Gebruik WA-IMPORT om een lijst uit WhatsApp te plakken.
EN: Go to SET > SET BEHEER. Use WA-EXPORT to copy list to clipboard. Use WA-IMPORT to paste a list from WhatsApp.

4. Live Mode

NL: Klik in de setlist op een titel voor Live Mode (XL letters + scroll).
EN: Tap a setlist title for Live Mode (XL text + scroll).
