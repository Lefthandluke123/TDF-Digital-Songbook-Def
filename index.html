Nederlands - Handleiding

1. Liedjes & Tags

Klik op SELECT om meerdere liedjes te kiezen. Vink ze aan en tik op een tag in de cloud om deze aan alle geselecteerde liedjes toe te voegen. Gebruik de (+) knop om nieuwe tags aan te maken.

2. Importeren & Backups

JSON Import: Gebruik dit voor volledige backups van je hele database. Op Android/iPad: kies 'Bestanden' als de foto-app opent.
TXT Import: Voor het toevoegen van losse liedjes. De bestandsnaam wordt de titel.

3. Setlists & Archief

Voeg liedjes toe aan je set met het plusje (+). Gebruik SAVE SET om je lijst op te slaan in het archief. Met WA-IMPORT kun je een lijst namen uit WhatsApp plakken om direct een set te bouwen.

4. Live Mode

In Live Mode verdwijnt de bovenbalk. De witte klok is de huidige tijd, de amberkleurige klok is de verstreken tijd van je optreden. Lettergrootte (A+) en BPM-snelheid worden per liedje onthouden.

English - Manual

1. Songs & Tagging

Click SELECT to pick multiple songs. Check them and tap a tag in the cloud to apply it to all selected items. Use the (+) button to create new tags.

2. Files & Backups

JSON Import: Best for full database backups. If Android/iPad opens the Photos app, select 'Files' instead.
TXT Import: Add single song files. Filename becomes the title.

3. Setlists & Archive

Add songs to your set with the (+) button. Use SAVE SET to store lists in your archive. WA-IMPORT allows you to paste names from WhatsApp to build a set instantly.

4. Live Mode

Top bar is hidden in Live Mode. White clock is current time, Amber clock is elapsed performance time. Font size (A+) and BPM speed are saved per song.
